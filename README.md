# 🍃 vAyu-QI

### Air Quality Intelligence System

vAyu-QI is an AI-powered environmental intelligence platform designed to predict Air Quality Index (AQI) levels and provide actionable environmental insights using Machine Learning.

The system combines AQI prediction, pollutant analysis, health impact assessment, environmental recommendations, and government initiatives into a single interactive platform.

---

## 🚀 Features

* AQI Prediction using Machine Learning
* Simple and Advanced Prediction Modes
* Intelligent Environmental Simulation
* Historical City Pattern Auto-Fill
* Health Impact Analysis
* Recommended Personal & Environmental Actions
* Government Initiative Awareness
* Dominant Pollutant Detection
* Pollutant Visualization Dashboard
* Interactive Streamlit Interface

## 🌍 Project Workflow

1. User selects prediction mode.
2. Pollutant values are entered or auto-filled using historical city patterns.
3. Machine Learning model predicts AQI.
4. AQI category is determined.
5. Health impacts and environmental recommendations are generated.
6. Dominant pollutant is identified.
7. Pollutant levels are visualized through charts.
